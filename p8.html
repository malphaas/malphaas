A simple dragon curve sequence program

The dragon curve is the curve you get when you repeatedly fold a strip of paper on itself always along the same direction with respect to the closed edge, then expand it so that all the folds are right angles. If we denote a right turn by 1 and a left turn by 0 the first few iterations of the sequence would be as follows (the sequence starts at one fold ie two edges): 

		1
		110
		1101100
		110110011100100
		1101100111001001110110001100100

...and so on. It can be seen that each iteration is simply the complemented mirror image of the previous iteration appended to itself with a 1 in between. Let our current index be i and let us assume it is not one of the nodal points with '1' value described. The situation is like this:

(0,1,2,...,2k-2)(2k-1)(2k,...,i,...,2k+1-2)

The position of i in the second mirror image half is (i-2k), thus its reflection in the first half would be at (2k-2) - (i-2k) = 2*(2k-1)-i where 2k-1 is of course the index of the nodal point. In the code below the running nodal index is denoted by the variable tk and n is the number of bits to generate. 

#include <stdio.h>

int main() {
	int i,tk=0,n=400;
	int d[1000];
	d[0]=1;

	printf("%d",d[0]);
	for(i=1;i<n;i++) {
		if(i==tk*2+1) { tk=i; d[i]=1; }
		else d[i]=!d[2*tk-i];
		printf("%d",d[i]);
	}

	printf("\n");
	return 0;
}
