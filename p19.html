A solution of the power sum system of order 3

Consider the system of equations,

x + y + z = a ...(1)
x2 + y2 + z2 = b ...(2)
x3 + y3 + z3 = c ...(3)

Or more succintly, Σx = a, Σx2 = b, Σx3 = c.

Squaring the first and subtracting the second, a2 - b = 2 Σxy or Σxy = (a2 - b) / 2. Using an identity,

Σx ( Σx2 - Σxy) = Σx3 - 3xyz
=> a ( b - (a2 - b)/2 ) = c - 3xyz
=> xyz = (a3 - 3ab + 2c)/ 6xyz
=> z = (a3 - 3ab + 2c)/ 6xy

Setting λ = x+y and μ = 2xy, and using the fact z = a - (x+y) from (1),

(a3 - 3ab + 2c)/ 6xy = a - (x+y)
=> 3(a - λ) μ = a3 - 3ab + 2c ...(4)

Again, considering (2),

x2 + y2 + z2 = b
=> (x+y)2 - 2xy + [a - (x+y)]2 = b
=> λ2 - μ + [a - λ]2 = b
=> μ = 2λ2 - 2aλ + a2 - b ...(5)

Substituting the value of μ from (5) into (4) we obtain a cubic equation in lambda,

3(a - λ) ( 2λ2 - 2aλ + a2 - b) = a3 - 3ab + 2c

Solving this equation then using the values of λ and μ found from (5) to solve the system λ = x + y, μ = 2xy for x and y, z naturally follows from z = a - (x+y).

Let us take a concrete example for clarity.

x + y + z = 6
x2 + y2 + z2 = 14
x3 + y3 + z3 = 36

Here a = 6, b = 14 and c = 36 thus the cubic equation for λ becomes 3 (1 - λ) ( 2λ2 - 2λ + 22 ) = 36 or λ3 - 12λ2 + 47λ - 60 = 0. The solutions for λ = 3,5,4. The solutions of λ and attendant values of μ, x,y, z as found by the above process are as follows:

λ = 3, μ = 4 => x,y = 1,2 or x,y = 2,1 and z = 3
λ = 5, μ = 12 => x,y = 2,3 or x,y = 3,2 and z = 1
λ = 4, μ = 6 => x,y = 1,3 or x,y = 3,1 and z = 2

Thus it generates x = 1, y = 2, z = 3 and permutations thereof which one may easily see satisfy the original system.
